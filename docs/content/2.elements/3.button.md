## Usage

Use the default slot to set the text of the Button.

::component-card
---
code: Button
---

Button
::

You can also use the `label` prop:

::component-card
---
props:
  label: Button
---
::

### Variant

Use the `variant` prop to change the visual style of the Button.

::component-card
---
props:
  variant: primary
baseProps:
  label: Button
---

::
### Size

Use the `size` prop to change the size of the Button.

::component-card
---
props:
  size: sm
baseProps:
  label: Button
---
::
