## Usage

::component-card
---
baseProps:
  name: 'radio'
---
::

### Label

Use the `label` prop to display a label on the right.

::component-card
---
baseProps:
  name: 'radio1'
props:
  label: 'Label'
---
::

### Required

Use the `required` prop to display a red star next to the label.

::component-card
---
baseProps:
  name: 'radio2'
props:
  label: 'Label'
  required: true
---
::

### Help

Use the `help` prop to display some text under the Radio.

::component-card
---
baseProps:
  name: 'radio3'
props:
  label: 'Label'
  help: 'Please choose one'
---
::

### Disabled

Use the `disabled` prop to disable the Radio.

::component-card
---
baseProps:
  name: 'radio4'
  value: true
props:
  disabled: true
---
::
