## Usage

Nuxt UI provides a Vue component [UApp](/components/app) for globally configuring locale and other settings.

```vue [app.vue]
<script setup lang="ts">
import { de } from '#ui/locale'
</script>

<template>
  <UApp :locale="de">
    <NuxtPage />
  </UApp>
</template>
```

::tip
Nuxt UI components are using English **by default**.
::
